家用伺服器折腾手记-3：Mac mini 部署 Homebridge 教程 🏠

发布时间：2025-03-21 | 分类：服务器, Docker, 智能家居

前言 📝

在前两篇文章中，我们已经在 Mac mini 上搭建了青龙面板和 MoviePilot。这一次，我们将探索如何使用 Homebridge 来打造智能家居中枢，让不支持 HomeKit 的设备也能接入苹果生态。

环境准备 🛠️

1. 安装依赖

brew install node

2. 创建工作目录

mkdir -p ~/docker/homebridge
cd ~/docker/homebridge

部署 Homebridge 🚀

1. 创建 docker-compose.yml

version: '3'
services:
  homebridge:
    image: oznu/homebridge:latest
    container_name: homebridge
    restart: unless-stopped
    network_mode: host
    environment:
      - TZ=Asia/Shanghai
      - PGID=1000
      - PUID=1000
      - HOMEBRIDGE_CONFIG_UI=1
      - HOMEBRIDGE_CONFIG_UI_PORT=8581
    volumes:
      - ./config:/homebridge

2. 启动服务

docker-compose up -d

基础配置 ⚙️

1. 访问控制面板

  • 打开 http://localhost:8581
  • 默认用户名：admin
  • 默认密码：admin

2. 添加插件

推荐安装以下插件：

  • homebridge-xiaomi-roborock-vacuum
  • homebridge-mi-air-purifier
  • homebridge-broadlink-rm

设备接入 🔌

1. 小米设备

{
    "platform": "MiRobotVacuum",
    "devices": [
        {
            "name": "扫地机器人",
            "ip": "192.168.1.100",
            "token": "your-device-token",
            "model": "roborock.vacuum.s7"
        }
    ]
}

2. BroadLink 遥控器

{

    "name": "客厅空调",

    "mac": "your-device-mac",

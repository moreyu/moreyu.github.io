    家用伺服器折腾手记-4：Mac mini 搭建 Homebridge 教程

    为什么选择 Homebridge？ 🤔

    • 支持多种智能家居设备：Homebridge 支持市面上大多数智能家居设备，包括但不限于灯光、插座、传感器、摄像头等。
    • 易于安装和配置：Homebridge 的配置非常简单，只需要在 Homebridge 配置文件中添加设备信息即可。
    • 强大的自动化能力：Homebridge 可以与其他自动化平台（如 IFTTT、Home Assistant）集成，实现更复杂的自动化控制。
    • 良好的兼容性：Homebridge 可以与多种智能家居平台（如 Apple HomeKit、Google Home）集成。

    环境准备 🛠️

    确保你的 Mac mini 上已经安装了 Docker 和 Docker Compose。如果还没有安装，可以参考我们的第一篇文章。

    部署步骤 📝

    1. 创建配置文件

    首先创建一个新的目录来存放 Homebridge 的配置文件：

    mkdir -p ~/docker/homebridge
cd ~/docker/homebridge

    创建 docker-compose.yml 文件：

    version: "3.8"
services:
  homebridge:
    image: homebridge/homebridge
    container_name: homebridge
    volumes:
      - ./config:/homebridge/config
    ports:
      - "8080:8080"
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=Asia/Shanghai
    restart: unless-stopped
    注意：请将 /path/to/media 替换为你实际的媒体文件路径。

    2. 启动服务

    在配置文件目录下运行：

    docker-compose up -d

    查看初始密码：

    docker exec -it homebridge ./homebridge

    3. 基础配置

    1. 访问 http://你的IP:8080
    2. 使用默认用户名 admin 和上一步获取的密码登录
    3. 进入管理面板，修改密码
    4. 添加设备：
      • 本地设备：直接添加设备信息
      • 云服务：配置各类云服务

    进阶配置 ⚙️

    1. 反向代理设置

    如果你想通过域名访问 Homebridge，可以配置 Nginx 反向代理：

    server {
    listen 80;
    server_name homebridge.yourdomain.com;

    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}

    2. SSL 配置

    建议配置 SSL 证书，确保数据传输安全：

    server {
    listen 443 ssl;
    server_name homebridge.yourdomain.com;

    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;

    # ... 其他配置同上
}

    3. 文件预览优化

    Homebridge 支持多种文件预览方式，可以根据需要进行配置：

    • 视频预览：支持 HLS 转码，平滑播放
    • 音频预览：支持在线播放和播放列表
    • 文档预览：支持 Office、PDF 等格式
    • 图片预览：支持图片缩放和幻灯片播放

    1. 缓存配置：
      environment:
  - CACHE_SIZE=1000

    2. 资源限制：
      services:
  homebridge:
    deploy:
      resources:
        limits:
          memory: 1G
        reservations:
          memory: 512M

    常见问题 ❓

    权限问题
    如果遇到文件访问权限问题，请检查：
    • 容器内的 PUID 和 PGID 设置
    • 挂载目录的权限设置
    • SELinux 配置（如果启用）

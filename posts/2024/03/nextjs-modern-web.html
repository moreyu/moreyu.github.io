引言

Next.js 13 新特性

Next.js 13 引入了多项重要更新，显著改变了应用的构建方式：

  • 基于文件系统的 App Router
  • 默认启用的 React Server Components
  • 改进的数据获取机制
  • 优化的图片组件

数据获取策略

Next.js 13 提供了多种数据获取方式，可以根据需求选择最适合的策略：

// 静态数据获取
export async function generateStaticParams() {
  const posts = await getPosts();
  return posts.map(post => ({
    slug: post.slug
  }));
}

// 服务器端数据获取
async function getData() {
  const res = await fetch('https://api.example.com/data', {
    next: { revalidate: 60 } // 每60秒重新验证
  });
  return res.json();
}
